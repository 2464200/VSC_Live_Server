GUIDA CONFIGURAZIONE MOBILE
===========================

Stato aggiornato:
- il progetto usa il server unificato sulla porta 5500
- non e' necessario usare "Live Server" o "Go Live"

PASSI CONSIGLIATI
-----------------
1. Apri la cartella `C:\VSC_Live_Server` in VS Code
2. Avvia il progetto con:

   powershell
   .\startup.ps1 -NoWait

3. Trova l'IP locale del computer con `ipconfig`
4. Sul telefono apri:

   http://[IP_DEL_PC]:5500/public/mobile.html

Esempio:
http://192.168.1.100:5500/public/mobile.html

NOTE IMPORTANTI
---------------
- computer e telefono devono stare sulla stessa rete WiFi
- la porta corretta e' 5500
- evita di usare "Open with Live Server" per queste pagine

SE NON FUNZIONA
---------------
1. verifica `http://localhost:5500/diagnostica.html`
2. verifica che `startup.ps1` abbia avviato il server
3. controlla eventuali firewall locali
