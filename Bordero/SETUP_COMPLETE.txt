## HOW TO USE THIS PROJECT

1. All files are in: /Bordero/ folder
2. Documentation files:
   - README.md - Full documentation
   - PROJECT_STATUS.md - Detailed status
   - REPORT_ESTRAZIONE_BORDERO.md - Feature analysis

3. Start from: index.html (opens in browser)
4. Use local HTTP server (not file://)
   
   python -m http.server 8000
   or
   npx http-server -c-1

5. Navigate to: http://localhost:8000/Bordero/
