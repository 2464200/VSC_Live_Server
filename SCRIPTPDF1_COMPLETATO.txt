🎯 COME USARE SCRIPTPDF1
═════════════════════════════════════════════════════════════════════════════════

1. ACCEDI ALLA PAGINA
   URL: http://localhost:8765/Prova/ScriptPDF1.html
   Apri il link nel browser

2. INTERFACCIA PRINCIPALE
   Titolo: "Gestione PDF - ScriptPDF1"
   
   Sezione 1: SELEZIONE FILE
   └─ ComboBox: "Seleziona un file PDF"
                (mostra lista da C:\SCRIPT_PDF)
   
   Sezione 2: VISUALIZZAZIONE INFO
   └─ Nome file corrente
   └─ Posizione nella lista (es: "1 di 2")
   
   Sezione 3: PULSANTI NAVIGAZIONE
   └─ ◀ Precedente  (vai al PDF precedente)
   └─ Successivo ▶  (vai al PDF successivo)
   └─ ✕ Chiudi      (chiudi Chrome e torna a Excel)

3. MODALITÀ DI APERTURA PDF

   Opzione 1: Click su "Successivo ▶"
   └─ Apre il PDF corrente in Chrome mentre naviga al successivo
   
   Opzione 2: Click su "Precedente ◀"
   └─ Apre il PDF corrente in Chrome mentre naviga al precedente
   
   Opzione 3: Doppio click sulla ComboBox
   └─ Apre il PDF selezionato senza cambiare file
   
   Opzione 4: Click sul nome del PDF (nella sezione info)
   └─ Apre il PDF corrente
   
   Opzione 5: Seleziona dalla ComboBox + doppio click
   └─ Seleziona il file che desideri + doppio click per aprire

4. VISUALIZZAZIONE PDF
   • Chrome si apre in modalità Kiosk (fullscreen, nessun menu)
   • Il PDF si visualizza in Chrome con Adobe Acrobat Reader integrato
   • Usa i controlli di Chrome per: zoom, pagine, stampa, scarica

5. CHIUSURA
   • Clicca il pulsante "✕ Chiudi"
   • Chrome si chiude automaticamente
   • Torna alla pagina HTML (o a Excel se integrata)

🛠️ TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

PROBLEMA: "Il PDF non si apre in Chrome"
SOLUZIONE:
  1. ✓ Assicurati che il server sia avviato:
     → cd C:\VSC_Live_Server && node pdf-server.js
  
  2. ✓ Verifica che il file esista:
     → explorer C:\SCRIPT_PDF
  
  3. ✓ Usa Alt+Tab per visualizzare Chrome (potrebbe essere nascosto)
  
  4. ✓ Controlla la console del browser (F12) per errori
     → Apri http://localhost:8765/Prova/ScriptPDF1.html
     → Premi F12 per aprire DevTools
     → Guarda la console per messaggi di errore
  
  5. ✓ Verifica che Chrome sia in esecuzione:
     → Apri Task Manager (Ctrl+Shift+Esc)
     → Cerca "chrome.exe"
  
  6. ✓ Test manuale dell'API:
     → Apri Windows PowerShell
     → Esegui:
        $result = Invoke-WebRequest -Uri "http://localhost:8765/api/pdf-list" -UseBasicParsing
        $result.Content | ConvertFrom-Json

PROBLEMA: "Chrome non trovato"
SOLUZIONE:
  1. ✓ Installa Google Chrome da https://www.google.com/chrome/
  2. ✓ Verifica percorso di installazione standard:
     C:\Program Files\Google\Chrome\Application\chrome.exe

PROBLEMA: "Nessun file PDF trovato nella cartella"
SOLUZIONE:
  1. ✓ Aggiungi file PDF in: C:\SCRIPT_PDF
  2. ✓ Filtra solo .pdf (non .txt, .doc, ecc)
  3. ✓ Ricarica la pagina HTML (F5)
  4. ✓ Riavvia il server

PROBLEMA: "La pagina HTML non si carica"
SOLUZIONE:
  1. ✓ Verifica che il server sia avviato (porta 8765)
  2. ✓ Prova il ping: http://localhost:8765/
  3. ✓ Controlla il firewall di Windows
  4. ✓ Usa un altro browser (Firefox, Edge)

✨ NOTE E SUGGERIMENTI
═════════════════════════════════════════════════════════════════════════════════

1. PERFORMANCE
   • Il primo lancio di Chrome potrebbe richiedere 2-3 secondi
   • Successivi lanci sono più veloci (cache di Chrome)
   • Per aperture rapide, mantieni un'istanza di Chrome in background

2. MONITOR SECONDARIO
   • Chrome si apre generalmente sul monitor secondario
   • Se hai un solo monitor, Chrome si apre su quello principale
   • Usa Alt+Tab per trovare la finestra se non la vedi

3. MODALITÀ KIOSK
   • Nessun menu (non si vede File, Edit, ecc)
   • Nessuna barra degli indirizzi
   • Schermo intero (F11 da qualsiasi finestra Chrome)
   • Premi ESC o clicca il pulsante Chiudi per uscire

4. INTEGRAZIONE EXCEL
   • Puoi lanciare da Excel usando VBA
   • Apri il file ScriptPDF1_VBA_Module.bas per l'integrazione

5. AGGIORNARE LISTA PDF
   • La lista si aggiorna automaticamente al caricamento pagina
   • Se aggiungi PDF mentre è aperto, ricarica con F5
